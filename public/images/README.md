# Photos des Créateurs

Placez vos photos de profil dans ce dossier avec les noms suivants :

- `creator1.jpg` - Photo du créateur 1
- `creator2.jpg` - Photo du créateur 2

## Formats supportés
Vous pouvez utiliser les formats suivants : `.jpg`, `.jpeg`, `.png`, `.webp`

Si vous utilisez un autre format ou nom de fichier, pensez à mettre à jour les chemins dans `src/pages/Creators.jsx` (lignes 18 et 32).

## Recommandations
- **Dimensions** : 500x500 pixels minimum (format carré recommandé)
- **Poids** : Moins de 500 Ko par image
- **Qualité** : Les images seront affichées en cercle, assurez-vous que le visage est bien centré
